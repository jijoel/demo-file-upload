Demo File Upload
======================
This is a demonstration of some file upload components, using [Laravel](https://laravel.com), [Vue](http://vuejs.org), [Vuetify.js](https://vuetifyjs.com), and [Vue-Clip](http://vueclip.adonisjs.com/).
